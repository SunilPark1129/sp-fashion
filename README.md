## Project Description

- It's a clothing web-store project that display various clothes.
- This website does not have implemented features such as login, purchase, or about contact etc pages.

## Installation
1. Clone the repo
```
git clone https://github.com/SunilPark1129/sp-fashion.git
```
2. Install
```
npm install
```
2. run the website
```
npm start
```
[Link](https://sip-clothing.com/)
